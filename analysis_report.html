Implied Volatility Smile Live Data

BS generates a flat smile at 20% (cannot capture skew). Heston with ρ=−0.7 generates a downward-sloping skew (leverage effect). Synthetic market has a U-shaped smile — both ITM and OTM options trade at elevated IV above ATM, typical of index options.

Key insight: The Heston model with ρ<0 reproduces a monotone downward skew (leverage effect: equity falls → vol rises). Real index options exhibit a U-shaped smile (volatility smile, both wings elevated), requiring extensions such as jumps (Bates model) or stochastic vol-of-vol (rough Heston) for a complete fit.

Implementation Highlights

Heston Characteristic Function (Albrecher 2007)
def _heston_cf(u, S0, r, T, v0, kappa, theta, xi, rho):
    """Branch-cut stable CF — works for scalar u (quad)
    and array u (FFT), including complex u for damping."""
    i = 1j
    xi_bar = kappa - xi * rho * i * u
    d = np.sqrt(xi_bar**2 + xi**2 * (u**2 + i * u))

    g = (xi_bar - d) / (xi_bar + d)
    exp_dT = np.exp(-d * T)

    C = kappa * theta / xi**2 * (
        (xi_bar - d) * T
        - 2.0 * np.log((1.0 - g * exp_dT) / (1.0 - g))
    )
    D = (xi_bar - d) / xi**2 * (
        (1.0 - exp_dT) / (1.0 - g * exp_dT)
    )
    return np.exp(i*u*(np.log(S0)+r*T) + C + D*v0)
Newton-Raphson Implied Volatility (tol=1e-6)
def implied_volatility(market_price, S, K, T, r,
                          tol=1e-6, max_iter=100):
    # Newton-Raphson: quadratic convergence
    # Bisection fallback: guarantees convergence
    sigma = 0.20
    low_sigma, high_sigma = 1e-4, 10.0

    for _ in range(max_iter):
        p, error = price(sigma), price(sigma) - market_price
        if abs(error) < tol: return sigma

        vega = _bs_vega_raw(S, K, T, r, sigma)
        sigma_new = sigma - error / vega  # Newton step

        if low_sigma < sigma_new < high_sigma:
            sigma = sigma_new
        else:                              # bisect fallback
            sigma = 0.5 * (low_sigma + high_sigma)

    warnings.warn("IV did not converge", RuntimeWarning)
